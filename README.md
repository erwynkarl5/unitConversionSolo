My Unit Conversion Solo project

**Description**

Unit Conversion Solo project allows user to input and convert values between different units of measurement, this project is designed to help beginners understand how JavaScript can handle user input, perform calculations, and update the DOM dynamically. It also demonstrates clean structuring of HTML, CSS, and JS files for small-scale applications.

Executing Program -Locate the main file: 'unitConversionSolo.html' -Right click the file and select 'open with' choose your web browser -The Unit Conversion webpage will load and display in the browser, users can input numerical values to convert. -If using VS code, you can install the Live server extension changes while editing
